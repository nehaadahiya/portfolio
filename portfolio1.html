Netflix Data Analysis Dashboard

A comprehensive data analysis to uncover Netflix content trends and support strategic decisions through visual storytelling and dashboards.

Problem

Difficulty understanding content trends across regions and genres due to scattered data and lack of visual clarity.

Solution

Built an interactive Tableau dashboard and Python EDA pipeline to consolidate insights, drive data-driven strategy, and improve content targeting.

Scope

Focused on Netflix’s 8,000+ titles dataset, genre popularity, regional distribution, and release patterns for strategic recommendations.

Tools & Skills Used

Python
Tableau
SQL
Excel

Challenges & Steps Taken

Challenges

  • Data privacy and cleaning inconsistencies.
  • High computational complexity with large data.
  • Lack of explainability in certain ML steps.

Steps Taken

  • Used efficient preprocessing and feature selection methods.
  • Implemented visual storytelling techniques to simplify complex patterns.
  • Focused on explainable AI elements to improve transparency.
